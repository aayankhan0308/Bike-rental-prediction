# 🚴 Bike Rental Demand Prediction

Predicting hourly bike rental demand using Machine Learning (Random Forest Regressor)

## 📊 Results
- **R² Score : 0.8614** (86.14% accuracy)
- **RMSE     : 240.29**

## 📁 Dataset
Seoul Bike Sharing Dataset — UCI ML Repository  
https://archive.ics.uci.edu/dataset/560

## 🛠️ Libraries Used
- pandas, numpy
- scikit-learn
- matplotlib, seaborn
- pickle

## 📂 Files
| File | Description |
|------|-------------|
| `eda.py` | Exploratory Data Analysis + graphs |
| `2_model.py` | ML model training and evaluation |
| `SeoulBikeData.csv` | Dataset |

## 🔍 Model
- Algorithm : Random Forest Regressor
- Train/Test Split : 80/20
- Trees : 100
